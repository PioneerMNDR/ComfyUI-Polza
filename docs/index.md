## Введение

**ComfyUI-Polza** — это пакет пользовательских нод для [ComfyUI](https://github.com/comfyanonymous/ComfyUI), который предоставляет удобный доступ к более чем 300 AI-моделям через единый API [Polza.ai](https://polza.ai).

### Возможности

- 💬 **Chat Completions** — работа с текстовыми моделями (GPT-4o, Claude, Gemini, DeepSeek и др.)
- 👁️ **Vision** — мультимодальный анализ изображений
- 🎨 **Text-to-Image** — генерация изображений через OpenAI-совместимый API
- 🖼️ **Media Image** — универсальная нода для работы с изображениями, видео, аудио и музыкой
- 📝 **Show Text** — утилита для отображения текста в графе

## Установка

### Через ComfyUI Manager (рекомендуется)

1. Откройте **ComfyUI Manager**
2. Перейдите в раздел **Install Custom Nodes**
3. Найдите **ComfyUI-Polza** в списке
4. Нажмите **Install**
5. Перезапустите ComfyUI

### Вручную

```bash
# Перейдите в директорию custom_nodes вашего ComfyUI
cd ComfyUI/custom_nodes

# Клонируйте репозиторий
git clone https://github.com/your-name/ComfyUI-Polza.git

# Перейдите в папку ноды
cd ComfyUI-Polza

# Установите зависимости
pip install -r requirements.txt
```

### Требования

- Python 3.8+
- ComfyUI (последняя версия рекомендуется)
- Доступ к интернету для работы с API

## Настройка API-ключа

Для работы с нодами необходим API-ключ от Polza.ai. Получите его на [polza.ai](https://polza.ai).

### Способ 1: Переменная окружения (рекомендуется)

```bash
# Linux/macOS
export POLZA_API_KEY="pk-your-key-here"

# Windows (PowerShell)
$env:POLZA_API_KEY="pk-your-key-here"

# Windows (CMD)
set POLZA_API_KEY=pk-your-key-here
```

### Способ 2: Файл конфигурации

Создайте файл `config.json` в папке ноды:

```json
{
  "api_key": "pk-your-key-here"
}
```

### Способ 3: Прямой ввод в ноде

Каждая нода имеет поле `api_key`, куда можно ввести ключ напрямую. Этот способ удобен для тестирования, но менее безопасен.

## Обзор нод

### 💬 Polza Chat

**Назначение:** Работа с текстовыми языковыми моделями (Chat Completions).

**Входы:**
- `api_key` (опционально) — API-ключ
- `model` — название модели (например, `openai/gpt-4o`)
- `prompt` — текстовый запрос
- `system_prompt` (опционально) — системная инструкция
- `temperature` (опционально) — температура генерации (0.0–2.0)
- `max_tokens` (опционально) — максимальное количество токенов

**Выходы:**
- `text` — сгенерированный текст

**Пример использования:**
```
model: openai/gpt-4o
prompt: "Объясни квантовую физику простыми словами"
temperature: 0.7
```

---

### 👁️ Polza Vision

**Назначение:** Мультимодальный анализ изображений (image + text → text).

**Входы:**
- `api_key` (опционально) — API-ключ
- `model` — модель vision (например, `openai/gpt-4o`)
- `image` — входное изображение
- `prompt` — вопрос или инструкция по анализу
- `max_tokens` (опционально)

**Выходы:**
- `text` — текстовое описание/анализ изображения

**Пример использования:**
```
model: openai/gpt-4o
image: [загруженное изображение]
prompt: "Опиши, что изображено на этой картинке"
```

---

### 🎨 Polza Text-to-Image

**Назначение:** Генерация изображений через OpenAI-совместимый API.

**Входы:**
- `api_key` (опционально) — API-ключ
- `model` — модель для генерации (например, `gpt-image-1`, `dall-e-3`)
- `prompt` — текстовое описание изображения
- `size` — размер изображения (например, `1024x1024`)
- `quality` — качество (`standard`, `high`)
- `n` (опционально) — количество изображений

**Выходы:**
- `images` — сгенерированные изображения

**Пример использования:**
```
model: gpt-image-1
prompt: "Космический пейзаж с планетами и звёздами"
size: 1024x1024
quality: high
```

---

### 🖼️ Polza Media Image

**Назначение:** Универсальная нода для работы с медиа: изображения, видео, аудио (TTS), музыка.

**Входы:**
- `api_key` (опционально) — API-ключ
- `model` — модель для генерации
- `prompt` — текстовое описание
- `image` (опционально) — входное изображение для image-to-image
- `aspect_ratio` — соотношение сторон (для изображений)
- `duration` — длительность (для видео)
- `voice` — голос (для TTS)
- `speed` — скорость воспроизведения (для TTS)
- `video_resolution` — разрешение видео
- `image_resolution` — разрешение изображения
- `strength` (опционально) — сила влияния при image-to-image (0.0–1.0)

**Выходы:**
- `images` — сгенерированные изображения
- `media_url` — URL на медиафайл (видео, аудио)

**Поддерживаемые типы контента:**

| Тип | Примеры моделей |
|-----|-----------------|
| 🖼️ Изображения | seedream-3, seedream-4-5, nano-banana, flux-1-1-ultra |
| 🎬 Видео | veo-3, veo-3-1, wan-2-6, kling-3-0, sora |
| 🔊 Аудио (TTS) | elevenlabs-tts-turbo |
| 🎵 Музыка | (доступные музыкальные модели) |

**Пример использования (изображение):**
```
model: seedream-3
prompt: "Киберпанк город ночью с неоновыми огнями"
aspect_ratio: 16:9
quality: high
image_resolution: 2K
```

**Пример использования (видео):**
```
model: veo-3-1
prompt: "Астронавт идёт по поверхности Марса"
duration: 10s
video_resolution: 1080p
```

**Пример использования (TTS):**
```
model: elevenlabs-tts-turbo
prompt: "Привет! Добро пожаловать в Polza AI."
voice: Rachel
speed: 1.0
```

### 📝 Polza Show Text

**Назначение:** Утилита для отображения текста прямо в интерфейсе ComfyUI.

**Входы:**
- `text` — текст для отображения

**Выходы:**
- `text` — тот же текст (для передачи дальше по цепочке)

**Пример использования:**
Полезно для отладки и просмотра промежуточных результатов в сложных воркфлоу.

## Примеры использования

### 1. Text-to-Image (базовый пример)

```
[🎨 Polza Text-to-Image]
  model: gpt-image-1
  prompt: "Космический пейзаж с планетами"
  size: 1024x1024
  quality: high
      ↓ images
[Preview Image]
```

### 2. Цепочка: Chat → Image

Генерация промпта через чат, затем создание изображения:

```
[💬 Polza Chat]
  model: openai/gpt-4o
  prompt: "Придумай детальное описание для картины в стиле киберпанк"
      ↓ text
[🎨 Polza Text-to-Image]
  model: gpt-image-1
      ↓ images
[Preview Image]
```

### 3. Vision → Image (описание и перерисовка)

```
[Load Image] → image → [👁️ Polza Vision]
                          prompt: "Опиши стиль этого изображения"
                              ↓ text
                        [🖼️ Polza Media Image]
                          model: seedream-3
                          prompt: ← (текст из Vision)
                              ↓ images
                        [Save Image]
```

### 4. Image-to-Image

```
[Load Image] → image → [🖼️ Polza Media Image]
                          model: seedream-3
                          prompt: "Сделай ярче, добавь закат"
                          strength: 0.7
                              ↓ images
                        [Preview Image]
```

### 5. Генерация видео

```
[🖼️ Polza Media Image]
  model: veo-3-1
  prompt: "Астронавт на Марсе, панорамный вид"
  duration: 10s
  video_resolution: 1080p
      ↓ media_url
[Download Video]
```

### 6. Text-to-Speech (TTS)

```
[🖼️ Polza Media Image]
  model: elevenlabs-tts-turbo
  prompt: "Привет! Добро пожаловать в Polza AI."
  voice: Rachel
  speed: 1.0
      ↓ media_url
[Download Audio]
```

## Технические детали

### Асинхронная генерация

Генерация изображений и видео может занимать значительное время. Ноды автоматически обрабатывают асинхронные запросы:

1. **Отправка запроса** — нода отправляет запрос к API
2. **Polling режим** — если ответ не готов мгновенно, запускается цикл проверки статуса
3. **Интервал проверки** — каждые 4 секунды
4. **Таймаут** — максимум 10 минут ожидания
5. **Загрузка результата** — при завершении результат скачивается и возвращается

Прогресс генерации отображается в логах ComfyUI.

### Обработка ошибок

Ноды обрабатывают следующие типы ошибок:

- **Неверный API-ключ** — выводится сообщение об ошибке
- **Таймаут** — если генерация превышает 10 минут
- **Недостаточно средств** — уведомление о балансе
- **Недоступность модели** — информация о статусе модели

### Производительность

- Время генерации зависит от модели и нагрузки на сервер
- Изображения: обычно 10–60 секунд
- Видео: от 1 до 10 минут в зависимости от длительности
- TTS: обычно несколько секунд

## Частые вопросы (FAQ)

### Q: Как получить API-ключ?
**A:** Зарегистрируйтесь на [polza.ai](https://polza.ai) и создайте API-ключ в личном кабинете.

### Q: Нода выдаёт ошибку "Invalid API key"
**A:** Проверьте, что:
- API-ключ указан правильно
- Ключ активен и не истёк
- На балансе есть средства

### Q: Генерация занимает слишком много времени
**A:** Это нормально для сложных моделей. Максимальное время ожидания — 10 минут. Если задача выполняется дольше, проверьте логи на наличие ошибок.

### Q: Можно ли использовать свои модели?
**A:** Ноды работают только с моделями, доступными через API Polza.ai. Список поддерживаемых моделей доступен на сайте.

### Q: Как обновить ноды?
**A:** 
```bash
cd ComfyUI/custom_nodes/ComfyUI-Polza
git pull
pip install -r requirements.txt
```
Затем перезапустите ComfyUI.

### Q: Поддерживается ли локальная установка?
**A:** Нет, ноды работают только через облачный API Polza.ai.
